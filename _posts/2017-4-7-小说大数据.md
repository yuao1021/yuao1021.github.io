> 使用 **tree** 在终端显示树状文件结构

![](https://ww4.sinaimg.cn/large/006tKfTcgy1fdhotefcb5j315s0ugjwk.jpg)

#### 安装 tree
使用 **brew** 进行安装

	$ brew install tree

#### 使用
- 直接使用 `tree` 命令，会在当前文件目录下，递归输出所有文件层级
	
		$ tree

- 限制层级

		$ tree -L 2

- 指定当前目录下的某个文件夹

		$ tree Desktop
	
#### 导出文件  
用`> 文件名.格式` 的形式导出

	$ tree -L 1 > tree.md
